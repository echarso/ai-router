# Price Runner Backend - Python Implementation

This is the Python implementation of the Price Runner backend. For architecture details and Python vs Go comparison, see the [Go backend README](../backend-go/README.md).

## Quick Start

```bash
cd backend-python
pip install -r requirements.txt
python app.py
```

## Differences from Go Implementation

- Uses Flask instead of Gin
- More concise code due to Python's syntax
- Easier integration with Python ML libraries
- Slower performance but faster development
